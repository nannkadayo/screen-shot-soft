# screen-shot-soft

## 概要
定期的にスクリーンショットを撮影し、指定されたサーバーに送信するツールです。

## 想定される適法な使用用途
- **従業員同意済みの業務監視**: 労働契約に明記され、従業員が事前に同意した上での業務時間中の作業記録
- **保護者による未成年の見守り**: 保護者が自身の監督下にある未成年の端末を管理する場合
- **自己管理・生産性向上**: 自分自身の作業記録や時間管理のため
- **セキュリティ監査**: 組織のIT管理者が、正式な権限に基づいて実施するセキュリティ監査

## 重要な使用条件・法的注意事項

### 必須要件
1. **明示的な同意取得**: 監視対象者から書面による事前同意を必ず取得してください
2. **通知義務**: 監視していることを明確に通知し、監視範囲・目的・保存期間を説明してください
3. **データ管理責任**: 個人情報保護法に準拠したデータ管理体制を構築してください
4. **アクセス権限**: 対象端末の正当な管理権限を有していることを確認してください

### 禁止事項
- 本人の同意なく第三者の端末にインストールすること
- 監視していることを隠蔽すること
- 業務に無関係な私的な時間・場面を監視すること
- 取得した情報を目的外で使用すること

### データ保護
- 収集したスクリーンショットは暗号化して保存してください
- アクセスログを記録し、不正アクセスを防止してください
- 必要な期間を超えたデータは速やかに削除してください
- 第三者への提供は法令に基づく場合を除き禁止です
