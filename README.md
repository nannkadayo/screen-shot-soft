# screen-shot-soft

## 概要
定期的にスクリーンショットを撮影し、サーバーに送信するシンプルなツールです。

## 想定用途
- 自分自身の作業記録・生産性分析
- 同意済みの従業員の業務記録（要事前通知・同意）
- 保護者による子供の端末管理
- プログラミング学習・研究用途

## 使い方
1. Node.jsサーバー（`index.js`）をセットアップ
2. クライアント（`Program.cs`）をビルド
3. 撮影間隔を設定
4. 使用前に`uploads/blank.md`は削除してください

## 注意
他者の端末に無断でインストールしないでください。
使用する場合は対象者への通知と同意が必要です。
